# Carpeta de Imágenes

Coloca aquí tus imágenes para el sitio web.

## Imagen de Perfil
- Nombre del archivo: `profile.jpg`
- Tamaño recomendado: 400x400 píxeles o mayor
- Formato: JPG, PNG o WebP

Una vez que agregues tu foto con el nombre `profile.jpg`, el sitio la mostrará automáticamente en lugar del ícono del astronauta.
